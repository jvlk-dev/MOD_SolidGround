# 🌏 Solid Ground

**Solid Ground** is a lightweight gameplay overhaul for Minecraft 1.20.1 Forge that fundamentally changes how you interact with the world. Inspired by the survival mechanics of games like *Valheim* and *TerraFirmaCraft*, this mod moves away from "instant-strip-mining" and encourages a slower, more deliberate approach to resource gathering.

## ⚒️ Core Features
*   **⚡ Mining Fatigue (Exhaustion):** Breaking blocks adds physical strain. Your fatigue builds up as you mine, gradually slowing your speed.
*   **📉 Depth-Based Scaling:** The deeper you go, the "denser" the world feels. Mining speed is dynamically scaled based on your Y-level.
*   **🏡 Builder-Friendly Bypass:** Any block placed by a player is exempt from all fatigue and depth penalties.
*   **📦 Server-Side Persistence:** Fatigue levels are saved to player NBT and synced across the network.

## 🧩 Compatibility
*   **Universal World Gen Support:** Fully compatible with mods like *Terralith* or *Tectonic*. Just add custom stone variants to the config whitelist.
*   **Tetra:** Fully compatible. Mechanics are block-bound, so modular tools work as intended.
*   **Create & Iron's Spells:** Encourages the use of machinery and magic as late-game mining solutions.

## ⚙️ Configuration
The mod is highly configurable via `solidground-common.toml`. You can adjust:
- Fatigue build-up and decay rates.
- Block whitelists.
- Depth-scaling multipliers and Y-level thresholds (Supports custom depths down to -2048).

## 📦 Modpack Permissions
You are free to include **Solid Ground** in any modpack, project, or server. No permission is required!
